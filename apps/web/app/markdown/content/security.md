# Security

A short summary of how we handle security today. The surface is small, and we want you to be able to read the whole page.

## Controls

- **HTTPS by default** — every site on Blode.md is served over HTTPS.
- **GitHub OAuth** — sign-in runs through GitHub OAuth. We request the minimum scopes needed.
- **Open source** — the code is public on GitHub. You can read every line.

## Reporting an issue

Found a problem? Email [m@blode.co](mailto:m@blode.co) with steps to reproduce. We will take it from there.
