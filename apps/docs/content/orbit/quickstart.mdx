---
title: Quickstart
description: Push docs content and attach it to the Orbit tenant.
---

1. Create or select the `orbit` project in the API.
2. Generate an API key for that project.
3. Push docs content with `blodemd push`.
4. Open `https://orbit.blode.md` to verify the deployment.

## Publish command

```bash
blodemd push apps/docs/content/orbit --project orbit --api-url https://api.blode.md
```
