---
title: docs.json reference
description: Configure navigation, branding, and runtime behavior with docs.json.
---

Every Blode.md project is configured through a single `docs.json` file at the root of your docs directory.

<Callout type="info">
  If you are coming from Mintlify, Blode.md intentionally keeps this contract
  smaller. `theme`, `colors`, `fonts`, `icons`, `background`, and `styling` are
  not supported in `docs.json`.
</Callout>

## Minimal example

```json title="docs.json"
{
  "$schema": "https://blode.md/docs.json",
  "name": "My Project",
  "slug": "my-project",
  "navigation": {
    "groups": [{ "group": "Getting started", "pages": ["index"] }]
  }
}
```

## Branded example

```json title="docs.json"
{
  "$schema": "https://blode.md/docs.json",
  "name": "My Project",
  "slug": "my-project",
  "description": "Documentation for my product.",
  "logo": {
    "light": "/logo/light.svg",
    "dark": "/logo/dark.svg",
    "alt": "Example Co. logo",
    "href": "https://example.com"
  },
  "favicon": "/favicon.svg",
  "appearance": { "default": "system" },
  "navbar": {
    "links": [{ "label": "GitHub", "href": "https://github.com/example/docs" }]
  },
  "navigation": {
    "tabs": [
      {
        "tab": "Guides",
        "groups": [
          { "group": "Getting started", "pages": ["index", "quickstart"] }
        ]
      }
    ]
  },
  "contextual": {
    "options": ["copy", "view", "chatgpt", "claude"]
  },
  "api": {
    "openapi": "openapi.yaml",
    "playground": { "display": "interactive" }
  },
  "seo": { "indexing": "all" }
}
```

## API reference

The `api` object controls OpenAPI and AsyncAPI rendering plus playground behavior.

<TypeTable
  type={{
    "api.openapi": {
      description:
        "Path to your OpenAPI spec. Accepts a string, array of strings, or a source object with `source`, `basePath`, `directory`, and `include`.",
      type: "string | string[] | OpenApiSource",
    },
    "api.asyncapi": {
      description: "Path to your AsyncAPI spec. Same format as `openapi`.",
      type: "string | string[] | OpenApiSource",
    },
    "api.playground": {
      description: "API playground settings.",
      type: "{ display?: 'auth' | 'interactive' | 'none' | 'simple', proxy?: boolean, credentials?: boolean }",
    },
    "api.examples": {
      description: "Autogenerated code example settings.",
      type: "{ autogenerate?: boolean, defaults?: 'all' | 'required', languages?: string[], prefill?: boolean }",
    },
    "api.params": {
      description: "Parameter display settings.",
      type: "{ expanded?: 'all' | 'closed' }",
    },
    "api.mdx": {
      description: "MDX API page settings.",
      type: "{ auth?: { method, name }, server?: string | string[] }",
    },
    "api.url": {
      description: "URL display mode for endpoint headers.",
      type: "'full'",
    },
  }}
/>

## Contextual actions

<TypeTable
  type={{
    "contextual.display": {
      description: "Where to render the action buttons.",
      type: "'header' | 'toc'",
    },
    "contextual.options": {
      description: "Array of built-in presets or custom action objects.",
      type: "Array<string | CustomOption>",
    },
  }}
/>

Built-in option presets: `copy`, `view`, `chatgpt`, `claude`, `perplexity`, `grok`, `aistudio`, `cursor`, `vscode`, `windsurf`, `devin`, `mcp`, `add-mcp`, `devin-mcp`, `assistant`.

## Validation

Run `blodemd validate` to check your `docs.json` against the schema before deploying:

```bash
blodemd validate
```
