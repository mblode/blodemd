title: docs.json reference
description: Complete reference for the docs.json configuration file.
---

Every Blode.md project is configured through a single `docs.json` file at the root of your docs directory. This page covers every top-level field.

## Minimal example

```json title="docs.json"
{
  "$schema": "https://mintlify.com/docs.json",
  "name": "my-project",
  "theme": "mint",
  "colors": { "primary": "#0D9373" },
  "navigation": {
    "groups": [{ "group": "Getting started", "pages": ["index"] }]
  }
}
```

## Full example

```json title="docs.json"
{
  "$schema": "https://mintlify.com/docs.json",
  "name": "my-project",
  "theme": "mint",
  "description": "Documentation for my product.",
  "colors": {
    "primary": "#0D9373",
    "light": "#CFF6EE",
    "dark": "#0C3A33"
  },
  "logo": {
    "light": "/images/logo-light.svg",
    "dark": "/images/logo-dark.svg"
  },
  "favicon": "/images/favicon.svg",
  "fonts": {
    "heading": "Inter",
    "body": "Inter",
    "provider": "google"
  },
  "navbar": {
    "links": [{ "label": "GitHub", "href": "https://github.com/example/docs" }],
    "primary": {
      "type": "button",
      "label": "Dashboard",
      "href": "https://app.example.com"
    }
  },
  "navigation": {
    "tabs": [
      {
        "tab": "Guides",
        "groups": [
          { "group": "Getting started", "pages": ["index", "quickstart"] }
        ]
      }
    ]
  },
  "appearance": { "default": "system" },
  "search": { "prompt": "Search docs..." },
  "seo": { "indexing": "all" },
  "metadata": { "timestamp": true },
  "contextual": {
    "options": ["copy", "view", "chatgpt", "claude"]
  },
  "api": {
    "openapi": "openapi.yaml",
    "playground": { "display": "interactive" }
  }
}
```

## Navbar

The `navbar` object configures the top navigation bar.

<TypeTable
  type={{
    "navbar.links": {
      description: "Array of link objects displayed in the navbar.",
      type: "Array<{ label: string, href: string }>",
    },
    "navbar.primary": {
      description: "Primary CTA button in the navbar.",
      type: "{ type: 'button' | 'discord' | 'github', label?: string, href: string }",
    },
  }}
/>

## Contextual actions

The `contextual` object adds action buttons to each page, letting readers copy content, open it in AI assistants, or perform custom actions.

<TypeTable
  type={{
    "contextual.display": {
      description: "Where to render the action buttons.",
      type: "'header' | 'toc'",
    },
    "contextual.options": {
      description: "Array of built-in presets or custom action objects.",
      type: "Array<string | CustomOption>",
    },
  }}
/>

Built-in option presets: `copy`, `view`, `chatgpt`, `claude`, `perplexity`, `grok`, `aistudio`, `cursor`, `vscode`, `windsurf`, `devin`, `mcp`, `add-mcp`, `devin-mcp`, `assistant`.

## Validation

Run `blodemd validate` to check your `docs.json` against the schema before deploying:

```bash
blodemd validate
```
