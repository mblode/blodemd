---
title: Wildcard domains
---

Wildcard domains let you automatically serve every subdomain from a single Vercel project.

## Requirements

Wildcard domains require using Vercel nameservers so Vercel can complete DNS challenges for SSL certificates.

- `ns1.vercel-dns.com`
- `ns2.vercel-dns.com`

## Setup

1. Point your apex domain to Vercel nameservers.
2. Add the apex domain in your Vercel project.
3. Add the wildcard domain entry: `.yourapp.com`.

Once configured, any subdomain (`tenant1.yourapp.com`, `docs.tenant1.yourapp.com`) routes to the same deployment and receives SSL automatically.
