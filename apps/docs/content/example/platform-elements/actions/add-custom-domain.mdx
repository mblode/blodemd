## Overview

Add Custom Domain is a server-side action that provisions custom domains on your Vercel project and checks their configuration status. It powers the DNS verification workflow for tenant-owned domains.

## Installation

<Installer path="add-custom-domain" />

## What it does

- Adds a domain to your Vercel project
- Returns DNS configuration requirements
- Surfaces verification state for your UI
